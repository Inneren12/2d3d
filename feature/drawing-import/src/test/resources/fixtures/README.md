# Golden Test Fixtures

This directory contains reference images for SSIM-based golden tests in the CV pipeline.

## Required Files

| File | Description | Source |
|------|-------------|--------|
| `test_blueprint_01.jpg` | Input blueprint photo (document on flat surface) | Captured manually |
| `edges_expected.png` | Reference Canny edge map for test_blueprint_01.jpg | Generated by EdgeDetector |
| `rectified_expected.png` | Reference rectified output for test_blueprint_01.jpg | Generated by PerspectiveWarper |

## Adding Fixtures

1. Capture or provide the input image (`test_blueprint_01.jpg`)
2. Run the CV pipeline in a known-good state to generate reference outputs
3. Verify outputs visually before committing
4. Commit to repository (use Git LFS for large images if needed)

## SSIM Thresholds

- Edge detection: SSIM > 0.95 vs `edges_expected.png`
- Warp/rectification: SSIM > 0.95 vs `rectified_expected.png`
- Blur detection: Pass/fail classification test

## Notes

- Images are binary-compared per sprint spec (file-level SHA256, not pixel-level)
- SSIM threshold of 0.95 may need tuning based on real-world testing (see sprint-2 Technical Debt)
- Do NOT commit PII or personally identifiable content in fixture images
