# Active Glossary

## Node (2d)
2D point with coordinates (x: Double, y: Double) in millimeters

## Member (2d)
Line between two nodes with profile reference

## Drawing (2d)
2D blueprint with collection of nodes and entities

## Scale (2d)
Mapping between pixel space and physical millimeters

## SSOT (global)
The single authoritative data source

## Determinism (global)
Property of a system to produce identical output for identical input

## DAG (global)
Graph structure for task dependencies without circular references
